The test case for the 1-my_list module
==========================================

the MyList is a class that inherits from Python list object.

Usage
======

Importing the add integer function from the module
    >>> MyList = __import__('1-my_list').MyList    

Creating the mylist object
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Empty list object
    >>> my_list = MyList()
    >>> print(my_list)
    []
    >>> my_list.print_sorted()
    []

Passing a list to mylist object
    >>> a = [2, 8, 5]
    >>> my_list = MyList(a)
    >>> print(my_list)
    [2, 8, 5]
    >>> my_list.print_sorted()
    [2, 5, 8]

Mixed sign numbers list
    >>> a = [-2, 3, 0, 10, -30]
    >>> my_list = MyList(a)
    >>> print(my_list)
    [-2, 3, 0, 10, -30]
    >>> my_list.print_sorted()
    [-30, -2, 0, 3, 10]

Same numbers list
    >>> a = [2, 2, 2, 2]
    >>> my_list = MyList(a)
    >>> print(my_list)
    [2, 2, 2, 2]
    >>> my_list.print_sorted()
    [2, 2, 2, 2]
    >>> my_list.append(2)
    >>> print(my_list)
    [2, 2, 2, 2, 2]
